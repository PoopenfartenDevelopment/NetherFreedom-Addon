## How to use
- Download the lastest Meteor Client build.
- Move NF Addon in your `.minecraft/mods` folder where you have installed Meteor.

## Features
- AutoWalkNF
- BaritoneMiner
- DiggingTools
- HotbarManager
- NetherBorer
- OffhandManager
- RotationNF
- Welcome HUD
